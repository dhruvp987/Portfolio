## Background
The web is very exciting. The world practically runs on it, after all! I think the greatest thing about it is that anyone can reach anyone or anything from anywhere. I could communicate with penguins living in Antartica, from the North Pole!

I made this website so that I could learn what it takes to make a good frontend, so that I could be able to contribute to the biggest and bestest things using the web.

I hope you enjoy using it as much as I enjoy making it!
